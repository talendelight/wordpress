## 1. Overview

### 1.1 Purpose
Create a public candidate submission form that allows candidates to submit their CV and contact information without requiring account creation or login.

### 1.2 User Role
**Candidate:** External individual submitting their own CV for job opportunities

### 1.3 Business Requirements
- No login required (friction-free submission)
- Capture minimal candidate data + CV file
- GDPR consent capture with timestamp
- Email confirmations to candidate and operators
- Generate unique CandidateID for tracking
- Store submissions in WordPress database

---

## 2. Page Details

### 2.1 URL
`/candidates-profile/`

### 2.2 Page Title
"Share Your Profile"

### 2.3 Page Layout
- Full-width Elementor template
- Form embedded in content section
- Minimal navigation (show header/footer)

## 3. Form Fields

### 3.1 Required Fields

| Field | Type | Validation | Notes |
|-------|------|------------|-------|
| First Name | Text | Required, min 2 chars | Given name |
| Last Name | Text | Required, min 2 chars | Family name |
| Email | Email | Required, valid email format | Primary contact method |
| Phone | Tel | Required, phone format | International format accepted |
| Privacy Policy Consent | Checkbox | Required (must be checked) | Links to privacy policy page |

**Conditional Requirement:** At least ONE of the following must be provided:
- LinkedIn Profile (URL, must include linkedin.com)
- CV/Resume (File Upload, PDF/DOC/DOCX only, max 10MB)
